## Prisma cmd
# It reads your Prisma schema (schema.prisma)
# Then it updates your database structure (tables, columns, etc.) to match the schema
# No migration files are created — it directly applies changes

npx prisma db push
# Generates Prisma Client from your schema
npx prisma generate –
# Updates your schema from the existing database structure
npx prisma db pull –
# Applies changes with a migration history
npx prisma migrate dev –
# ensures your generated client matches the latest schema
npx prisma generate
